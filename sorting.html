Sorting



What types of sorting?



In Java, there are a countless number of ways one could sort a list of items. In this tutorial, I will introduce 3 different ways to sort a list of values. These 3 techniques are called selection sort, merge sort, and radix sort. As well, I will show each sort's pros and cons, as well as when they should be used.


For the sake of a tutorial, I will only be explaining things for sorting into ascending order. Lets start!



Selection Sort



In selection sort, the list of values are repeatedly looped through to find the minimum value in the unsorted part, putting it into the beginning. It is important to remember that during this sort, there will be two subarrays constantly changing: the sorted subarray, and the remaining parts of the array to be sorted.


Let's take a look at the code implementation. I will show you what each line means: 

          
            public class Selectionsortdemo{

              public void sort(int [] array){
                for(int i = 0; i < array.length-1; i++){  // traverse through array

                  int minimumIndex = i;       // location of the current smallest element in the unsorted array
                  for(int j = i+1; j < array.length-1; j++){  // loops through the unsorted part of the array, starting at index i+1
                    if (array[j] < array[minimumIndex]){    // if a smaller value than our current minimum is found,
                      minimumIndex = j;                     // we set the value of our new current minimum's index to j
                    }
                  }

                  // swap the local minimum in the unsorted part with the element at i.
                  int temp = array[minimumIndex];
                  array[minimumIndex] = array[i];
                  array[i] = temp;
                }
              }

              public static void main(String [] args){
                int [] list1 = {0,-10,100,20,53,10,100};
                sort(list1);

                for(int element: list1){
                  System.out.println(element);
                }
              }
            }
